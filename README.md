 # Data Analyst Job Market Analysis

### Project Overview

This project analyzes the data analyst job market to identify key trends, in-demand skills, and salary drivers. Using SQL, I explored job posting data to answer critical career development questions for aspiring and current data analysts.

SQL queries can be found here: [project_sql folder](/project_sql/)

### Objectives

1. Identify top-paying data analyst roles

2. Determine essential skills for high-paying positions

3. Analyze market demand for specific skills

4. Understand the relationship between skills and salaries

5. Recommend optimal skills for career advancement

## Key Findings

**High-Demand Core Skills:**

- **SQL** appears in nearly all top-paying roles (8/10 jobs).

- **Python** is essential for 70% of high-paying positions.

- **Tableau** for data visualization (60% of top roles).

- **R** remains relevant for statistical analysis roles.


**Top-Paying Skill Categories:**

1. Data Engineering Tools: PySpark ($208k), Databricks ($142k), Airflow ($126k)

2. DevOps Integration: Bitbucket ($189k), GitLab ($154k), Jenkins ($125k)

3. AI/ML Platforms: Watson ($160k), DataRobot ($155k)

4. Cloud Technologies: GCP ($122k), Kubernetes ($132k)

5. NoSQL Databases: Couchbase ($160k), Elasticsearch ($145k)


**Optimal Skills (High Demand + High Salary):**

Skills that balance market demand with premium compensation include:

**Python libraries** (Pandas, NumPy)

**Cloud platforms** (GCP, AWS)

**Data visualization** tools (Tableau, Power BI)

**DevOps tools** (GitLab, Jenkins)

## Career Recommendations

1. Foundation First: Master SQL, Python, and Tableau as baseline requirements.

2. Specialize Upward: Add data engineering skills (Spark, Airflow) for salary growth.

3. Cloud Proficiency: Develop expertise in cloud platforms (GCP, AWS, Azure).

4. DevOps Integration: Learn version control and CI/CD tools for hybrid roles.

5. Strategic Learning: Focus on skills with both high demand and high pay.
